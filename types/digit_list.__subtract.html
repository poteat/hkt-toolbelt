Type alias _$subtract<A, B>

_$subtract<A, B>: _$trim<_$subtract2<A, B>>

_$subtract is a type-level function that subtracts one digit list from another. It returns the result of the subtraction.

Type Parameters

  • A extends DigitList

    A digit list representing a number to subtract.

  • B extends DigitList

    A digit list representing a number to subtract by.

Example

For example, we can use _$subtract to subtract one digit list from another:

import { DigitList } from "hkt-toolbelt";

type Result = DigitList._$subtract<["1", "2", "5"], ["1", "2", "1"]>;
// ["4"]

In this example, Result is a type that represents ["4"], which is the result of subtracting ["1", "2", "5"] from ["1", "2", "1"].
