Type alias _$equalsAll<T, PREV>

_$equalsAll<T, PREV>: T extends [infer CURR, ...(infer REST)]

        : _$equalsAll<REST, CURR>

_$equalsAll is a type-level function that takes in an array of types T, and returns true if all elements of T are equal or T is empty.

Type Parameters

  • T extends List.List

    An array of types.

  • PREV = T[0]

Template: U

A type.

Example

For example, we can use _$equalsAll to determine whether a series of types are equal. In this example, [false, false, false] is passed as a type argument to the type-level function:

import { Conditional } from "hkt-toolbelt";

type Result = Conditional._$equalsAll<[
 false,
 false,
 false,
]>; // true
