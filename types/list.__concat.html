Type alias _$concat<U, T>

_$concat<U, T>: U extends unknown[]
    ? [...T, ...U]
    : _$push<U, T>

_$concat is a type-level function that concatenates two tuples.

It takes in two arguments: T, the tuple to concatenate onto, and U, the tuple to concatenate.

Type Parameters

  • U extends unknown

    A tuple type to concatenate, or an unknown.

  • T extends unknown[]

    A tuple type to be concatenated onto.

Returns

A tuple type.

If U is not a tuple type, it will be pushed into T as its new last element.

Basic Usage

Example

type Result = List._$concat<[0, 1], [2, 3]>; // [0, 1, 2, 3]

## Advanced Usage

Concatenating to a tuple with a rest parameter results in a tuple that contains the concatenated tuple.

Example

type Result = List._$concat<[1, 2, ...string[]], ["foo"]>; // [1, 2, ...string[], "foo"]
