_$divide is a type-level function that performs the division operation. It takes in two integers A and B representing the dividend and divisor respectively, and returns the result of dividing A by B.

  • QUOTIENT_LIST extends DigitList = _$divide<A_LIST, B_LIST>

Returns

An integer type.

If A is not a multiple of B, the quotient is returned and the remainder is thrown away. The quotient is truncated towards zero. That is, -1/2 is evaluated to be 0, not -1.

Example

For example, we can use _$divide to divide 10 by 2:

import { Integer } from "hkt-toolbelt";

type Result = Integer._$divide<10, 2>; // 5

Example

If A is not a multiple of B, the result is truncated towards zero, and only the quotient is returned.

import { Integer } from "hkt-toolbelt";

type Result = Integer._$divide<-100, 99>; // -1
