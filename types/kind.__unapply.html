Type alias _$unapply<K, F, X>

_$unapply<K, F, X>: X

_$unapply is a type-level function that takes in a) a type-level function that has been partially or fully applied with an argument, b) the original type-level function that was invoked to derive the first input type, and then extracts and returns the applied argument from closure.

Type Parameters

  • K extends Kind

    The target type-level function to unapply.

  • F extends Kind

    The type-level function that was applied to an argument to derive K

    Note that _$unapply infers the most specific type for the closure argument, whereas Kind._$inputOf returns the widest category of arguments accepted by the input function.

  • X extends _$inputOf<F> = K extends $<F, infer X>
        ? X
        : never

Example

For example, we can use _$unapply to determine what argument was passed into an applied type-level function. In this example, we pass in Add2, the target function, and NaturalNumber.Add into _$unapply, which gives us the input that was passed into NaturalNumber.Add to derive Add2.

import { $, Kind, NaturalNumber } from "hkt-toolbelt";

type Add2 = $<NaturalNumber.Add, 2>
type Is2 = Kind._$unapply<Add2, NaturalNumber.Add> // 2
