Type alias _$decrementTens<A>

_$decrementTens<A>: _$decrementTens_LUT[A]

_$decrementTens is a type-level function that takes a single decimal digit type A, and determines whether the tens digit should be decremented in a subtraction operation. If decrementing the provided digit would require a carry-over from the tens digit (i.e., when A is "0"), the function returns "1". In all other cases, the function returns "0".

This function is intended to be used as part of a larger subtraction operation.

It only operates on individual digits and does not handle the logic for the full subtraction or decrementing of the tens digit.

Type Parameters

  • A extends Digit

    A one-character decimal digit type.

Example

For example, forwarding a decimal digit 9 will result in:

import { Digit } from "hkt-toolbelt"

type Result = Digit._$decrementTens<"9"> // "0"
