Type alias _$repeat<T, N, RESULT>

_$repeat<T, N, RESULT>: RESULT

_$repeat is a type-level function that returns a tuple filled with multiple elements of a specified type.

It takes in two arguments: T, the type to repeat, and N, the number of times to repeat T.

_$repeat can handle an output tuple length of up to 2137, which is larger than 999, the maximum recursion depth limit of TypeScript.

Type Parameters

  • T extends unknown

    An unknown type.

  • N extends Number

    A natural number.

  • RESULT extends List.List = _$isNatural<N> extends true
        ? _$repeat2<T, _$toList<N>>
        : never

Returns

A list of types containing N counts of T.

If N is not a natural number, returns never.

Example

type Result = List._$repeat<"a", 3>; // ["a", "a", "a"]
