Interface Pair

Pair is a type-level function that generates a tuple of pairs from a tuple, where each element is paired with the next element.

Template: T

The tuple to generate pairs from.

Returns

A list of tuples of length 2 that contain sequential elements of T.

Example

type T0 = $<List.Pair, [1, 2, 3, 4]> // [[1, 2], [2, 3], [3, 4]]
type T1 = $<List.Pair, []> // []
type T2 = $<List.Pair, [1]> // []
type T3 = $<List.Pair, [1, 2]> // [[1, 2]]
