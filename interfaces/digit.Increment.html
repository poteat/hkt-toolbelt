Interface Increment

Increment is a type-level function that takes in a digit type A and returns the next digit in the sequence. If the input digit is "9", the function returns "0".

Template: A

A digit type.

Example

For example, we can use Increment to increment a digit type:

We apply Increment to a digit type using the $ type-level applicator:

import { $, Digit } from "hkt-toolbelt"

type Result = $<Digit.Increment, "3"> // "4"
interface Increment {}
